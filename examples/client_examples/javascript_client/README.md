# In-Browser Training Example

Quickstart

1. Install  
*** Note if `npm setup` ran from top level of repo please skip step 1***
```console
npm install 
```

2. Run Example
```console
bash run_example.sh 3 <user_id> average_layers http://host.docker.internal:8000 http://0.0.0.0:8080
```

3. Run Client
```console
ts-node client.ts
```
